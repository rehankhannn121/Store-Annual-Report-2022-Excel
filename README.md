# Store Annual Report 2022 (Excel Dashboard)

## Project Overview
This project analyzes store sales data for the year 2022 to understand customer behavior, sales trends, channel performance, and regional distribution.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers
- Data Cleaning & Formatting 

## Key Insights
- Women contributed 64% of total sales.
- Amazon was the top-performing sales channel (36% of total sales).
- Mid-age customers generated the highest number of orders.
- Maharashtra and Karnataka were the highest revenue-generating states.
- Returns and refunds remained low, indicating strong customer satisfaction.



## Business Recommendations
- Focus marketing efforts on female customers and mid-age groups.
- Increase advertising spend on Amazon.
- Optimize product categories based on state-level performance.
- Analyze male product returns to reduce future refunds.
